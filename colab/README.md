# Colab package

Arquivos:
- `VaiiixBRxAprende_Colab.ipynb`: notebook para baixar os resumos do GitHub, treinar e subir artefatos.
- `VaiiixBRxAprende.py`: módulo-base do aprendizado, para referência e reaproveitamento.

Antes de rodar:
1. Abra o notebook no Google Colab.
2. Defina `GITHUB_TOKEN`, `GITHUB_REPO_OWNER`, `GITHUB_REPO_NAME` e `GITHUB_REPO_BRANCH`.
3. Rode as células em ordem.
4. Os artefatos serão publicados em `vaiiixbr/artifacts/`.
